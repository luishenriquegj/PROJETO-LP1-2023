add() é a função template responsavel por inserir um novo \mbox{\hyperlink{classNode}{Node}} na \mbox{\hyperlink{classList}{List}},este novo node deve vir populado com suas informações (data); para manter a reusabilidade é imporante utilizar o nome dos atributios de maneira identica aos usados na classe \mbox{\hyperlink{classMusic}{Music}}.
